github登录异常

技巧 github 2021/04/09

本文于 344 天之前发表，文中内容可能已经过时。

github上不去，大家都知道这是咋回事，本博主也是查看了网上各位大神的方法，按照操作后可以能够访问，故在此分享给大家。

无法链接原因：DNS污染，幸好不是背墙。程序员捣鼓一下还能上，这样应该主要是为了过滤小白。

具体步骤如下：

  1. 登录ping检测网站，查看github被解析的哪个ip能够被访问
  2. 修改本机的host,将对应的DNS解析添加进去
  3. 通过命令行刷新dns

1.登录ping检测网站

登录github检测网站，通过该网站查看对应的github网站被解析正常的ip，将红色框选部分的IP Address给复制下来备用

登录图片

2.将DNS解析添加到host文件中

host的目录为C:\Windows\System32\drivers\etc下的host文件，将其用文档编译器打开（注意：需要管理员模式打开，否则编辑后无法保存)，将之前的ip添加在最后面。ip和github.com需要注意添加空格

3.通过命令行刷新dns

win+R，输入cmd回车，随后输入命令ipconfig /flushdns

完成以上操作就可以访问Github了，以下是登录成功的截图
